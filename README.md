# Amdox Data Analytics Internship Projects

## Overview

This repository contains the projects completed during my Data Analytics Internship. The work focuses on applying data analytics, machine learning, and business intelligence techniques to solve real-world problems across retail, healthcare, and financial domains.

## Projects Included

### 1. NeuralRetail Analytics System

A retail analytics platform developed to analyze customer purchasing behavior, sales trends, customer segmentation, and churn risk.

**Key Features**

* Exploratory Data Analysis (EDA)
* Customer Segmentation (RFM Analysis)
* Churn Analysis
* Sales Trend Analysis
* Interactive Streamlit Dashboard
* Business Insights and Recommendations

**Technologies Used**

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Streamlit

---

### 2. Healthcare Intelligence & Medicine Recommendation System

A healthcare analytics solution designed to analyze patient data, assess risk levels, and provide medicine recommendations through an interactive dashboard.

**Key Features**

* Data Preprocessing
* Disease Analytics
* Risk Assessment
* Medicine Recommendation Module
* Interactive Healthcare Dashboard
* Health Score Calculator

**Technologies Used**

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn
* Streamlit

---

### 3. Financial Fraud Detection System

A machine learning-based fraud detection project developed to identify suspicious transactions and support financial risk management.

**Key Features**

* Fraud Detection Model
* Exploratory Data Analysis
* Transaction Pattern Analysis
* Fraud Risk Visualization
* Interactive Streamlit Dashboard

**Technologies Used**

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn
* Streamlit
